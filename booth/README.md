# 이니티움 실시간 구현소 — 갤러리 프로토타입

소학술제 부스용 "오늘의 제작 갤러리" 1차 버전입니다. 의존성 설치 없이 Node 내장 모듈만으로 동작합니다.

## 실행 방법

```bash
cd booth
node server.js
```

기본 포트는 3000번입니다 (`PORT=4321 node server.js`처럼 바꿀 수 있습니다).

- 운영 콘솔(구현자용, 관객에게 노출 X): http://localhost:3000/console.html
- 갤러리(관객용, iPad/QR로 보여줄 화면): http://localhost:3000/gallery.html

콘솔에서 추가하면 갤러리가 2.5초마다 자동으로 새로고침되어, 새로 만든 프로토타입이 폰 화면 모양으로 실시간으로 나타납니다. 투표 템플릿은 실제로 집계되고, 기기당 1회만 투표할 수 있도록 localStorage로 제한합니다.

## 지금 버전의 한계 (다음에 다듬을 것)

- 데이터는 `data/apps.json` 파일에 저장됩니다. 로컬 테스트용이며, 실제 Vercel에 배포할 때는 Supabase나 Vercel KV 같은 진짜 DB로 바꿔야 합니다 (서버리스 환경에서는 파일 쓰기가 유지되지 않음).
- 템플릿별 전용 입력 폼(혼잡도 슬라이더, 체크리스트 항목 등)은 아직 없고, 지금은 제목+한줄요약(또는 투표 선택지)만 받습니다.
- 지도 템플릿의 학교 평면도 배경 그래픽은 아직 없습니다.
