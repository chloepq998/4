# 우리 반, 너의 선택은?

학교 속 사회문제(방관자 효과·따돌림·배제)를 다루는 인터랙티브 선택형 게임입니다.
소학술제 전시를 위해 만든 데모로, 빌드 과정 없이 브라우저에서 바로 실행됩니다.

## 어떻게 동작하나요

- 6개의 교실 속 상황이 차례로 등장하고, 각 상황마다 **8초** 안에 선택해야 합니다.
- 선택에 따라 **용기 게이지(점수)** 가 오르내리고, 시간이 다 되면 가장 소극적인 선택이 자동으로 적용됩니다.
- 마지막에는 누적 점수에 따라 4가지 결과 중 하나가 나오고, 방관자 효과에 대한 짧은 설명과 실천 팁이 함께 제시됩니다.
- 결과 화면의 "함께 이야기해보기" 버튼을 누르면 부스에서 관람객과 나눌 토론 질문 화면으로 이동합니다.

## 실행 방법

별도 설치나 빌드가 필요 없습니다.

1. `index.html` 파일을 더블클릭해서 브라우저로 열거나,
2. 폴더에서 간단한 정적 서버를 띄워서 접속하세요.

```bash
python3 -m http.server 8000
# 이후 브라우저에서 http://localhost:8000 접속
```

## 파일 구성

- `index.html` — 화면 구조 (시작 / 게임 / 결과 / 토론)
- `style.css` — 디자인 (반응형, 모바일·태블릿 지원)
- `script.js` — 게임 로직 (시나리오 데이터, 타이머, 점수, 결과 분기)

## 시나리오 수정하기

`script.js`의 `SCENARIOS` 배열에 상황과 선택지를 자유롭게 추가·수정할 수 있고,
`RESULTS` 배열에서 점수 구간별 결과 문구를 조정할 수 있습니다.
